# Table of irregular verbs

- Present: (*I*) *am*, (*she, he, it*) *is*, (*you*, *we*, *they*) *are*
- Past: (*I, she, he, it*) *was*, (*you*, *we*, *they*) *were*
- *\-ed* form: *been*
